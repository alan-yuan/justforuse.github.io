## How to use
Your json.file should include `name` and `children`, like this:

```
{
    "name": "中国",
    "children": [
        {
            "name": "浙江",
            "children": [
                {
                    "name": "杭州"
                },
                {
                    "name": "宁波"
                },
                {
                    "name": "温州"
                },
                {
                    "name": "绍兴"
                }
            ]
        }
        {
            "name": "黑龙江",
            "children": [
                {
                    "name": "哈尔滨"
                },
                {
                    "name": "齐齐哈尔"
                },
                {
                    "name": "牡丹江"
                },
                {
                    "name": "大庆"
                }
            ]
        }
    ]
}
```
